# Blog Post App

## A MERN stack project (Mongo, Express, React, Node.js)

[Heroku deployment - demo -](https://mern-bb-blog.herokuapp.com)

Development

```
yarn run dev
```

Production build client + Start server

```
yarn start
```

### FOR DEVELOPMENT

install dependecies

```
yarn install
```

in mongoDB

```
use mernapp
```

[locally open http://localhost:8000/ -->](http://localhost:8000)
